Order list and Unorder List

Order List

OL = Order List
LI = List Item, obrigatoriamente é um complemento da OL. A partir do HTML5 o /li é opcional
Existem 4 tipos de ordenamento, para acessá-los basta inserirmos a função type dentro da tag de abertura OL. Também se quisermos iniciarmos a lista por um determinado elemento utilizamos o start após o type.
Exemplos:

  1. Arroz
  2. Feijão
  3. Açúcar
  4. Macarrão
  5. Fubá
  6. Pão

Unorder list

UL = Unorder List / Lista Não Ordenada.

  • Feijão
  • Açúcar
